# Find the Greatest Number  

This Python program takes three numbers as input and prints which one is the greatest.  

## How it works  
1. The program asks the user to enter three numbers.  
2. It compares them using if-else statements.  
3. It prints which number is the largest.  

## How to run  
1. Make sure Python is installed on your system.  
2. Download the file `self practise.py`.  
3. Open a terminal in the same folder and type:  
   python "self practise.py"
